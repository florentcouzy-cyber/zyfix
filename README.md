# ZyFix v2.0 — PWA Gestion Réparation PC

Application web progressive (PWA) tout-en-un pour la gestion d'un atelier de réparation informatique.

## Fichiers
- `index.html` — Application complète (SPA)
- `manifest.json` — Configuration PWA
- `sw.js` — Service Worker (cache offline)
- `icon-192.svg` — Icône 192×192
- `icon-512.svg` — Icône 512×512

## Nouveautés v2.0
- ☎️ Numéros de téléphone cliquables (`tel:`) — appel direct depuis mobile
- 📧 Emails cliquables → ouvre Gmail Nouveau message directement
- 📍 Adresse client dans prise d'appel + mise à jour fiche client

## Déploiement GitHub Pages
Activer GitHub Pages sur la branche `main` → l'app est accessible directement via l'URL.

Comment fonctionne l'e-Facture B2B dans ZyFix ?
L'e-Facture n'est pas une facture différente — c'est votre facture habituelle, enrichie d'une transmission électronique obligatoire vers les clients professionnels.
Le flux est : ① Créer un client pro (SIRET requis) → ② Cocher "Soumis e-Facture" → ③ Créer la facture normalement → ④ Elle apparaît ici → Préparer → Envoyer
1
Préparer le client pro
Dans l'onglet Clients, ouvrez la fiche d'un client Entreprise et remplissez le bloc "Facturation électronique" : SIRET, plateforme de réception, adresse électronique. Cochez "Soumis e-Facture".
2
Créer la facture
Créez la facture exactement comme d'habitude dans l'onglet Factures. Si le client est pro et soumis e-Facture, elle sera automatiquement détectée comme e-Facture B2B.
3
Transmettre
La facture apparaît dans le tableau ci-dessous. Cliquez 📋 Préparer pour générer le payload JSON, puis 📤 Envoyer pour la transmettre à la plateforme (disponible en 2027).

Configurer Google Drive Sync
📋 Configuration (5 min, gratuit)
  1. Va sur console.cloud.google.com
  2. Crée un projet → active Google Drive API
  3. Identifiants → Créer des identifiants → ID client OAuth
  4. Type : Application Web
  5. Origines autorisées : ajoute http://localhost et l'URL de ton hébergement
  6. Copie l'ID client ci-dessous
Tes données sont stockées dans l'espace privé de ton compte Google (appDataFolder), invisibles dans Google Drive.
